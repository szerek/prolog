Вышеупомянутая процедура состоит из трех частей:

Факты: опишите взаимосвязь между аптеками (идентификатор, название сети, адрес, телефон) и лекарствами (идентификационная карта, имя) и продажами (номер аптеки, номер лекарства, цена, сколько осталось).
Правила: Описывает, в каких аптеках продается лекарство и как найти самую дешевую аптеку для лекарства.
Запросы: Вы можете запросить адрес и номер телефона аптеки в зависимости от названия и количества препарата.
В приведенной выше программе факт аптека/4 описывает основную информацию об аптеке, факт лекарства/2 описывает основную информацию о лекарстве, а факт продажи/4 описывает информацию о лекарстве, продаваемом аптекой. Правило available_at/4 используется для определения наличия лекарства в аптеке. Правило самая дешевая аптека/4 используется для поиска самой дешевой аптеки для определенного лекарства. Начните запрос с ?-, и вы сможете запросить адрес и номер телефона аптеки по наименованию и количеству препарата.
